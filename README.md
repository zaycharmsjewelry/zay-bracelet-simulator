# ZAY Bracelet Simulator

This folder is ready for GitHub Pages.

## Upload to GitHub
1. Create a new public repository on GitHub.
2. Upload **everything inside this folder** (`index.html`, `style.css`, `script.js`, `charms.js`, and the `assets` folder).
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `main` and `/ (root)`, then Save.
6. GitHub will give you a public link you can place in your Instagram bio.

## Notes
- The simulator contains the two bracelet images you supplied.
- The charm sheets were automatically split into individual charm images.
- You can click a charm more than once.
- Drag charms on the bracelet to reposition them.
- "Save image" downloads the finished design as a PNG.
